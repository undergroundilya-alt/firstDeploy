# 01. Product scope beta

Цель beta-версии — показать не просто overlay от AdBlock, а SaaS-платформу, которая помогает владельцу сайта контролировать рекламную видимость перед выдачей защищённого контента.

## Что входит в beta

1. Аккаунт владельца продукта.
2. Компании-клиенты.
3. Проекты клиентов, где один проект обычно равен одному сайту.
4. Public key для подключения client SDK.
5. Secret key для server-to-server проверки.
6. Client SDK, который клиент вставляет на сайт.
7. Проверка рекламного контейнера.
8. Loader перед выдачей контента.
9. Overlay при подтверждённой проблеме рекламной видимости.
10. Мягкая классификация: вероятная блокировка / вероятная проблема соединения / DOM tamper.
11. Analytics dashboard.
12. Demo customer page.

## Что не нужно обещать в beta

- 100% невозможность обхода.
- Полную защиту от всех пользовательских фильтров.
- Точную финансовую потерю клиента без доступа к его рекламной статистике.
- Юридическую гарантию соответствия всем рекламным сетям.

Правильная формулировка: beta помогает измерять и контролировать рекламную видимость, снижать простой обход и тестировать связку ad visibility -> content access.
