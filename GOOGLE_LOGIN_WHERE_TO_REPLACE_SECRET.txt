GOOGLE LOGIN SETUP

1. Open the .env file in the project root.

2. Find this line:
   GOOGLE_CLIENT_SECRET=REPLACE_WITH_NEW_GOOGLE_CLIENT_SECRET

3. Replace only REPLACE_WITH_NEW_GOOGLE_CLIENT_SECRET with your NEW Google Client Secret.

Example:
   GOOGLE_CLIENT_SECRET=GOCSPX-your-new-secret-here

Important:
- Do not use the old secret that was pasted into chat.
- Rotate/create a new Client Secret in Google Cloud first.
- Keep GOOGLE_CLIENT_ID as it is unless you create a new OAuth client.

Google Cloud settings for local testing:
Authorized JavaScript origins:
   http://localhost:3443

Authorized redirect URIs:
   http://localhost:3443/auth/google/callback

Run locally:
   npm install
   npm start

Open:
   http://localhost:3443/login
